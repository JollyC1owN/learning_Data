reduce

概念

reduce() 方法对数组中的每个元素执行一个由您提供的回调函数(升序执行)，将其结果汇总为单个返回值。

语法

arr.reduce(callback(accumulator, currentValue, index, array), initialValue)
callback：回调函数
accumulator：累计器—个人可以理解为上一次回调的返回值（第一次除外）
currentValue：当前值—个人可以理解为数组中的每一项
index：当前索引
array：源数组
initialValue:作为第一次调用 callback函数时的第一个参数的值
返回值：函数累计处理的结果

回调函数第一次执行时，accumulator 和currentValue的取值有两种情况：

  • 如果调用reduce()时提供了initialValue，accumulator取值为initialValue，currentValue取数组中的第一个值；

  • 如果没有提供 initialValue，那么accumulator取数组中的第一个值，currentValue取数组中的第二个值。

注意：

如果没有提供initialValue，reduce 会从索引1的地方开始执行 callback 方法，跳过第一个索引。如果提供initialValue，从索引0开始。

如果数组为空且没有提供initialValue，会抛出TypeError（类型错误） 。如果数组仅有一个元素（无论位置如何）并且没有提供initialValue， 或者有提供initialValue但是数组为空，那么此唯一值将被返回并且callback不会被执行。

reduce()如何执行

1、无initialValue值

2、有initialValue值

[0, 1, 2, 3, 4].reduce(function (accumulator, currentValue, currentIndex, array){

      return accumulator + currentValue;

      }, 10 );

常用的几个例子

例子1-数组里所有值的和

计算数组里所有值的和

  let arr = [1,9,2,8,3,7,4,6,5,5];

  //accumulator 累计器

  //currentValue 当前值

  //currentIndex 当前索引

  //array 数组

  let sum = arr.reduce(function (accumulator,currentValue,currentIndex,array) {

    console.log(accumulator,currentValue,currentIndex)

    return accumulator+currentValue

  });

  console.log(sum) //结果------50

箭头函数的形式

let sum2=[0, 1, 2, 3, 4].reduce((prev, curr) => prev + curr ,10)

console.log(sum2) //结果------20

例子2-累加对象数组里的值

累加对象数组里的值
注意：

要累加对象数组中包含的值，必须提供初始值，以便各个item正确通过你的函数。

let arr = [{x:1},{x:2},{x:3}]

  let sum = arr.reduce(function (acc,curr){

    console.log("累计器："+acc,"当前值："+curr)

    return acc + curr.x

  },0)

  console.log(sum)

结果

例子3-将二维数组转化为一维

将二维数组转化为一维
初始值initialValue，可以写一个空数组 [ ]；也可以不写

 let arr = [[0,1],[2,3],[4,5]]

  let newArr = arr.reduce(function (acc,curr){

    console.log("累计器："+acc,"当前值："+curr)

    return acc.concat(curr)

  })

  console.log(newArr)

例子4-计算数组中每个元素出现的次数

计算数组中每个元素出现的次数
注意：初始值initialValue，必须有；为一个空对象 { }

let name = ["Tom","Tomb","kobe","Tomb","Tom","wade","kobe","Tomb"]

  let result = name.reduce(function (sumResult,name){

    console.log("累计器："+sumResult,"当前值："+name)

    if(name in sumResult){

      sumResult[name] +=1

    }else{

      sumResult[name] = 1

    }

    return sumResult

  },{})

  console.log(result)

结果

例子5-数组去重

数组去重
注意：初始值initialValue，必须有；为一个空数组 [ ]

let arr = [1,10,5,6,4,5,7,8,9,3,2,1,10,6,7]

  //调用sort()；对源数组进行排序（阿斯科码排序）结果：[1, 1, 10, 10, 2, 3, 4, 5, 5, 6 , 6, 7, 7, 8, 9]

  //用reduce()方法进行去重

  let newArr = arr.sort().reduce(function (everyResult,item){

    console.log("累计器："+everyResult,"当前值："+item)

    //用新数组 即everyResult,其中的最后一项与当前值进行比较，若不相等则添加到数组的最后

    if(everyResult[everyResult.length-1] !==item){

      everyResult.push(item)

    }

    return everyResult

  },[])

  console.log(newArr)

结果
