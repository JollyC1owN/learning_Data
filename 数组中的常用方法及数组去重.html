slice( )——截取数组中某一段

作用：通过起始位置和结束位置的索引，截取数组中某一段数组，包前不包后
参数：

  • 两个参数：arr.slice(m,n)； 截取m到n，包含m，不包含n

  • 一个参数：arr.slice(m)； 从m开始截取到最后一项

  • 0个参数：arr.slice( )： 截取整个数组，即：克隆数组，等价于arr.slice( 0 )

  • 注：参数支持负数，负数会默认加上数组的length后处理，或者理解为从后往前计数

返回值：截取的那段数组
注：原数组不发生改变

splice( )——删除数组的某一段

作用：从某一个索引开始删除n项，还可以在删除的位置添加新的内容
参数：

  • 一个参数：arr.splice( n )； 从n项开始，删除到数组的最后一项

  • 两个参数：arr.splice( m , n )；从m项开始，删除n项

  • 多个参数：arr.splice( m , n ，”a”,”b”)；从m项开始，删除n项，将后面的参数添加到删除的位置

  • 删除数组的最后一项：arr.splice(arr.length-1 , 1)

  • 在数组最后添加内容：arr.splice(arr.length , 0 , “last” , “gg”)

返回值：被删除的那一段
注：原数组发生改变

reduce

概念

reduce() 方法对数组中的每个元素执行一个由您提供的回调函数(升序执行)，将其结果汇总为单个返回值。

语法

arr.reduce(callback(accumulator, currentValue, index, array), initialValue)
callback：回调函数
accumulator：累计器—个人可以理解为上一次回调的返回值（第一次除外）
currentValue：当前值—个人可以理解为数组中的每一项
index：当前索引
array：源数组
initialValue:作为第一次调用 callback函数时的第一个参数的值
返回值：函数累计处理的结果

回调函数第一次执行时，accumulator 和currentValue的取值有两种情况：

  • 如果调用reduce()时提供了initialValue，accumulator取值为initialValue，currentValue取数组中的第一个值；

  • 如果没有提供 initialValue，那么accumulator取数组中的第一个值，currentValue取数组中的第二个值。

注意：

如果没有提供initialValue，reduce 会从索引1的地方开始执行 callback 方法，跳过第一个索引。如果提供initialValue，从索引0开始。

如果数组为空且没有提供initialValue，会抛出TypeError（类型错误） 。如果数组仅有一个元素（无论位置如何）并且没有提供initialValue， 或者有提供initialValue但是数组为空，那么此唯一值将被返回并且callback不会被执行。

reduce()如何执行

1、无initialValue值

2、有initialValue值

[0, 1, 2, 3, 4].reduce(function (accumulator, currentValue, currentIndex, array){

      return accumulator + currentValue;

      }, 10 );

    }

  },[])

  console.log(newArr)

双循环去重——continue法

        outer : for(let i=0; i<ary.length; i++){ //给for循环起一个名字

            for(let j=0; j<ary1.length; j++){

                if(ary[i] == ary1[j]){ //判断两个数组是否有重复项

                }

            ary1.push(ary[i])
